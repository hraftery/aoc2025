# aoc2024

Solutions for [AdventOfCode2025](https://adventofcode.com/2025).

---

To build all days:

```sh
make
```

To build day N:
```sh
make dayN
```

To run both parts of day N (1 through 12) on the real input:

```sh
build/dayN
```

To run part P (1 or 2) of day N on the real input:

```sh
build/dayN P
```

To run part P of day N on the example input X (a through z):

```sh
build/dayN P X
```
